# Análise de Dados – Viagens Governamentais

Projetos de análise de dados usando Python e Pandas
com dados públicos do Portal da Transparência.

## Tecnologias
- Python
- Pandas

## Mini-projetos
- Top 10 viagens mais caras
- Média e viagens acima da média
- Gastos por cargo

## Objetivo
Praticar análise exploratória de dados e
desenvolver portfólio para vagas iniciais.

⚠️ O dataset completo não foi incluído por ser muito grande.
Foi utilizada uma amostra aleatória de 5.000 registros.
